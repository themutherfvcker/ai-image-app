# Examples folder

Place real before/after image pairs here for the homepage slider.

Naming convention:
- <slug>-before.jpg
- <slug>-after.jpg

Where <slug> is one of:
- remove-background
- change-clothing-color
- product-ad
- storyboard
- age-transformation
- style-transfer

- Recommended aspect ratio: 16:9 (e.g., 1600x900)
